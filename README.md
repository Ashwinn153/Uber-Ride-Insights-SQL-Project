## Uber Ride Insights: Navigating Mobility Data

## Project Overview

This project analyzes a ride-hailing dataset inspired by Uber operations using SQL.
The analysis focuses on rides, drivers, passengers, payment methods, ride trends, and business insights.

# Dataset Used

The project contains 3 tables:

1. Rides

Contains ride-related information:

- ride_id
- driver_id
- passenger_id
- pickup & dropoff locations
- ride distance
- fare amount
- payment methods

2. Drivers

Contains driver details:

- driver_id
- driver_name
- rating
- earnings
- total rides

3. Passengers

Contains passenger details:

- passenger_name
- total rides
- total spent
- ratings

# SQL Concepts Used

- SELECT
